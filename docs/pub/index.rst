Python Users Berlin (PUB) is a diverse community around the Python open-source
programming language. Through outreach, workshops, events and social gatherings,
we aim to promote the Python community in Berlin. Anyone interested in Python is
welcome to join!

.. _pub-content:

Everyone is welcome to suggest topics they would like to learn and present.
More information can be found on our :doc:`wish-list`.
